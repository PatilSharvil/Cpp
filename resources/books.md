# 📚 Books for Systems Programming

## Core Books

### 1. Unix Network Programming by W. Richard Stevens
- **Volume 1**: Sockets and XTI (THE Bible for network programming)
- **Volume 2**: Interprocess Communication
- **Why**: Deep dive into sockets, TCP/IP, IPC
- **Read**: Start with Volume 1, Chapters 1-10 for HTTP server

### 2. Effective Modern C++ by Scott Meyers
- **Why**: Learn C++11/14/17 best practices
- **Read**: Items 1-10 (type deduction, smart pointers)

### 3. C++ Concurrency in Action by Anthony Williams
- **Why**: Master threads, mutex, atomics
- **Read**: Chapters 1-5 for thread pool

### 4. Computer Systems: A Programmer's Perspective (CS:APP)
- **Why**: Understand how code maps to hardware
- **Read**: Chapters on I/O, networking

## Reference Books

### 5. TCP/IP Illustrated by W. Richard Stevens
- Deep protocol understanding

### 6. The Linux Programming Interface by Michael Kerrisk
- Comprehensive system call reference

### 7. Designing Data-Intensive Applications by Martin Kleppmann
- System design principles
